# 🩺 NEET PG 2026 — Mission: Sub 1000

> A pair of interactive study tools built with love for a future doctor's 90-day sprint to NEET PG 2026.

No backend. No login. No installation. Just open in a browser and start studying.

---

## 📁 What's Inside

### 1. `neet-pg-tracker.html` — The Study Tracker
A clean, elegant progress dashboard for serious preparation.

**Features:**
- 📅 Live countdown timer to exam day
- 📊 Subject-wise topic checklist across all major NEET PG subjects
- ✅ Mark topics as done — progress saves automatically in the browser
- 📚 Curated study resources (YouTube, QBanks, Mock Tests) for each subject
- 🔍 Overall progress bar and per-subject completion percentage
- 🌙 Dark mode UI, no distractions

**Subjects covered:** Pharmacology · Anatomy · FMT · Pathology · Microbiology · Medicine · Surgery · OBG · Paediatrics · Psychiatry · PSM · and more

---

### 2. `NEET_PG.html` — Mission: Sub 1000 (Gamified Edition)
A full video-game style study companion. Same tracker, but make it fun.

**Features:**
- ⚡ **XP & Levelling System** — 10 levels from *MBBS Recruit* to *NEET PG Champion*
- 🏆 **18 Achievements** — unlock badges for streaks, accuracy, topics completed, and more
- 🎯 **MCQ Battle Log** — log daily MCQ attempts and track accuracy over time
- 📈 **Stats Dashboard** — XP earned, current level, best accuracy, topics studied
- 🔔 **Level-Up Animations** — satisfying popups when you level up
- 💾 **Auto-save** — all progress stored in browser localStorage, nothing is lost
- 📱 **Mobile friendly** — works on phone too

## 🚀 How to Use

1. **Download** either `.html` file (or both)
2. **Open** in any modern browser (Chrome, Firefox, Edge, Safari)
3. That's it — no internet needed after opening, no account, no app

> 💡 **Tip:** Bookmark it or pin the tab so you can come back to it every day. Your progress is saved in the browser automatically.

---

## 🔒 Privacy

All data stays on **your device only**. Nothing is sent anywhere. Progress is stored in your browser's `localStorage` — clearing browser data will reset it, so don't do that accidentally!
